# ACG-manager

A(ainme)・C(comic)・G(game)  
見たいアニメ・読みたいマンガ・したいゲームを管理するTodoアプリです

## 機能一覧
- Todoの一覧機能
- Todoの新規投稿機能
- Todoの編集機能
- Todoの削除機能

## 開発環境
- Ruby 
- Ruby on Rails
- javascript
- create-react-app
- MySQL
- GitHub
- Visual Studio Code
